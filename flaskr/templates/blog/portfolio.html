Portfolio

I'm in my last semester studying Computer Science at the University of Massachusetts, Amherst. During my time at the university I've tried to learn as much as possible and become a well-rounded software engineer. I've done a good mix of systems, web programming, and theory, and have worked on a whole lot of projects. I've listed some of them here.

Selected School Projects

  • CS377 Honors Operating Systems: I extended the kernel of the [XV6 operating system]() to support multi-threading. This included implementing the ['clone()']() and ['join()']() system calls, as well as ticket locks. This was a semester-long project which I did in order to help the professor develop new course content--it was part of the Honors section. I got really comfortable working in the operating system kernel!

  • CS403 Robotics: Wrote code for a simulated robot to play [Pong](). I implemented a hierarchical behavior in C. My undergraduate Honors thesis is extending that work in order to implement further sophisticated robot behavior.

  • CS446 Search Engines: Built a search engine from scratch. Later on I extended the search engine code to index blog posts on this website and have it working--try out the search bar above!

Work Experience

I have spent three summers in computer science internships. This past summer (2019), I made significant contributions to two M.I.T. Lincoln Laboratory projects. The first was a satellite communications module, for which I was given ownership over a big chunk of the software design and implementation. I wrote close to 2000 lines of C code for the project, and it was a fantastic learning experience for me. I learned a great deal about embedded systems programming, realtime operating systems, and communication protocols. I had to make decisions about memory/resource constraints, meeting real-time constraints, and leveraging hardware to squeeze the most performance out of the system. My second project was building a procedural mesh component for an orbit simulation tool developed at Lincoln Laboratory. The simulation tool was written using the Unity game engine, and I wrote an algorithm that could build a 3D model on-the-fly.
