Portfolio

I received my BS in Computer Science at the University of Massachusetts Amherst, and am currently in the final year of my Master's in Software Engineering at the Technical University of Munich. This page is all about the projects I've done for fun or for school; please see my LinkedIn profile for information about my professional experience.

I have made almost all of my code open source. Unfortunately, some projects were closed source.

Stefan on Software (2017-)

I've been working on this website on and off since 2017, when I decided to try making something with Python Flask. I developed the site layout with Bootstrap, which makes it possible to adjust how the website is displayed based on the size of the screen. I'm no UX designer, but I do my best to make this site reasonably easy on the eyes.

I host this website on PythonAnywhere and post articles via an API I developed. I write posts in Markdown, which then gets rendered to HTML as part of that upload process. I also wrote a search engine and a web service to analyze site traffic to go along with it.

Read the blog post or see the code.

Site Analytics (2020-2022)

Site Analytics is a Python Flask service I developed to record and analyze the web traffic I receive on this website. I use PostgreSQL for data storage and SQLAlchemy as an ORM

The service uses an API to record traffic, and a separate API to query results. Using the Query API, I posted an analysis of more than 30,000 requests received by my site over the past two years. I was able to do interesting things like see the geographic distribution of "bots" viewing my site and determine the operating systems used by people viewing my site

Read the blog post or see the code.

Benchmarking Caching Technologies for Ephemeral Workloads (2022)

I did a research project at the Technical University of Munich to test the performance of various caching technologies for workloads of ephemeral data. I developed a C++ benchmarking program capable of simulating various workloads, and tested four different technologies on a compute cluster using Docker. I wrote a paper about the results.
