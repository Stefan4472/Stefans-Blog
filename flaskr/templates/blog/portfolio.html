Selected School Projects

  • Developing Sophisticated Robot Behavior Using Hierarchical Behavior and Reinforcement Learning (Honors Thesis)

    I am working with Professor Roderic Grupen to complete my undergraduate thesis in Robotics. I am developing a "Fetch" action, in which a robot searches its surroundings for a ball, moves to it and picks it up, and brings it to a specific position. The idea is to implement this behavior using an action hierarchy, which is learned using Reinforcement Learning. It is an ongoing project, and I have learned much about robotics and RL. The code is in C, and will eventually use ROS (Robot Operating System).

  • Kernel Hacking (CS377 Honors Operating Systems)

    I extended the kernel of the XV6 Operating System, a Unix-like operating system developed at M.I.T. for teaching. My work included understanding the source code, adding system calls and structures to support multi-threading, and implementing synchronization mechanisms. This work was done in C. I also wrote a Python autograder script which could run automated tests in the XV6 environment--this is now being used by the professor to grade student projects in upcoming classes. I gained a detailed understanding of operating systems.

  • Shakespeare Search Engine (CS446 Search Engines)

    I built a search engine in Python which provided search capabilities over the entire corpus of Shakespeare's work. This gave me an understanding of how search engines work, and much of the theory they operate on. It was also a good architectural experience for me. Later I extended the search engine code to work for blog posts on this website--try out the search bar above!

  • UMass Outdoor Club Website (CS326 Web Programming)

    With a small group of other students, I developed a website for the UMass Outdoor Club using the Django Python web framework. My work included designing the database model, writing the user profiles, comment sections, and trip signups, and much of the back-end work. I learned how to use Django and develop a sophisticated web app, complete with user accounts and data.
