This page features some of the projects I've done for fun and for school; please see my LinkedIn profile for information about my professional experience.

Stefan on Software (2017-)

I've been working on this website on and off since 2017, when I decided to give Python Flask a try. I developed the site layout with Bootstrap, which provides some good-looking components and responsively adjusts to different screen sizes. I'm certainly not a UX designer, but I do my best to make this site look reasonably good.

I host this website on PythonAnywhere and post articles via an API I developed. I write posts in Markdown, which then gets rendered to HTML as part of that upload process. I also wrote a search engine and a web service to analyze site traffic (see below) as partner projects to this website.

Follow these links to read the blog post or see it on Github.

Site Analytics (2020-2022)

Site Analytics is a Python Flask service I developed to record and analyze the web traffic I receive on this website. I use PostgreSQL for data storage and SQLAlchemy as an ORM. By geolocating IP addresses and analyzing user agents I can find out interesting facts about the people and robots who visit my site.

The service uses an API to record traffic and a separate API to perform analytical queries. Using the Query API, I analyzed more than 30,000 requests received by my site over the past two years.

Follow these links to read the blog post or see it on Github.

I did a research project at the Technical University of Munich to test the performance of various cloud-based caching technologies. I developed a C++ benchmarking program capable of simulating various workloads and used Docker to test four different technologies on a compute cluster.

Bavarian Election System (2020)

Working with another student, I developed an electronic voting system for the Bavarian parliamentary elections. We developed our backend with Python Flask and PostgreSQL, performed load-testing with Locust, and wrote our frontend with React and Bootstrap. Our system accurately calculated the results of the 2013 and 2018 elections, processing over 25 million voting records.

Follow these links to read the blog post or see it on Github.
